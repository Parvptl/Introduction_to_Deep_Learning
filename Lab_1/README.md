## 🚀 Overview

This lab implements core classification algorithms **from scratch** using only NumPy and basic Python.

Implemented models:
- Binary Perceptron  
- Binary Logistic Regression  
- Multi-class Perceptron (One-vs-Rest)  
- Multi-class Logistic Regression (One-vs-Rest)  

---

## ⚙️ Key Concepts

- Linear decision boundaries  
- Gradient descent optimization  
- Binary Cross-Entropy loss  
- Feature standardization  
- One-vs-Rest multi-class strategy  
- Evaluation using Accuracy, Precision, Recall, F1-score  

---

## 📊 Highlights

- Hyperparameter tuning (learning rate, epochs)  
- Decision boundary visualization  
- Confusion matrix and per-class metrics  
- Comparison of generalization performance  

---

## 🎯 Outcome

This lab builds a strong foundation in linear classification models and demonstrates how classical ML algorithms form the basis of modern deep learning systems.
