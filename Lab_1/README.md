# Lab 1 – Perceptron & Logistic Regression (From Scratch)

**Course:** DS3040 – Introduction to Deep Learning  
**Tools:** Python, NumPy, PyTorch, Matplotlib  

## Objective
Implement Perceptron and Logistic Regression from scratch for binary and multi-class classification without using built-in sklearn models.

## Tasks
- Data preprocessing for binary and multi-class datasets
- Manual implementation of:
  - Perceptron
  - Logistic Regression (Sigmoid & Softmax)
- Training with different learning rates and iterations
- Decision boundary visualization
- Model evaluation on test set

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score

## Key Concepts
Linear classifiers, gradient-based optimization, cross-entropy loss, decision boundaries, generalization, hyperparameter tuning.

## Learning Outcome
Developed strong intuition for how linear classifiers learn, update weights, and generalize to unseen data.
